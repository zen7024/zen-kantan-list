# zen-kantan-list 🛒

お母さんのための買い物リストアプリです。
シンプルで使いやすいことを最優先に開発しました。

## できること
- 買うものをサクッと追加
- スーパー・ドラッグストアなど店舗別に分類
- 購入予定日の設定
- お気に入り登録でよく買うものをすぐ追加
- 購入済みチェック＆履歴管理
- カテゴリ（店舗名）を自由にカスタマイズ

## 対象ユーザー
スマホに不慣れな方でも直感的に使えるよう、
シンプルなUIを心がけました。
